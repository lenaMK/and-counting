# *[...] and counting*

2023, Lena MK

*[...] and counting* is a participatory mapping installation exploring how artworks by women artists entered Tiohtià:ke · Montréal's public space.



## Data
- contour de l'île de Montréal: simplification des données d'[Open Street Maps ](https://www.openstreetmap.org/relation/3430571)
- geojson [réseau cyclable de l'agglomération de Montréal](https://www.donneesquebec.ca/recherche/dataset/vmtl-pistes-cyclables)
- données MONA: extrait des données de l'API MONA, préparées dans le notebook [Chronologie et genre](https://observablehq.com/@maison-mona/chronologie-et-genre)
